# Как запускать это говно

 * Ставим [golang](https://go.dev/doc/install)
 * Ставим [Make](https://gnuwin32.sourceforge.net/packages/make.htm) если винда
 * Выполняем в консоли `make build`
 * Скачеваем из моей телеги архив с песнями
 * Выполняем в консоли `./main.exe match Em C G D` или на какие там вообще аккорды вам надо найти песню
